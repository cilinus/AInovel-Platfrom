# AI 소설/만화 서비스 플랫폼 기획서

## 📋 프로젝트 개요

AI 기술을 활용하여 소설과 만화 콘텐츠를 생성하고 사용자에게 서비스하는 플랫폼입니다.

### 핵심 특징
- **AI 생성 콘텐츠**: 소설, 만화를 AI가 자동 생성
- **사용자 맞춤형**: 개인화된 스토리 경험 제공
- **토큰 기반 결제**: 무료 토큰 → 유료 전환 계획
