## 📱 주요 화면 목록

### 화면 구성 개요

```
┌─────────────────────────────────────────┐

## 1️⃣ 온보딩 화면

### 1.1 스플래시
```
┌─────────────────────┐
│                     │
│                     │
│      [로고]         │
│                     │
│   "무한한 이야기"    │
│                     │
│                     │
│    ○ ○ ○ (로딩)    │

- 앱 로고 + 슬로건
- 자동 진행 (2초)

### 1.2 로그인/회원가입

│      ← 뒤로         │
├─────────────────────┤
│                     │

│                     │
│ ┌─────────────────┐ │
│ │ 📧 이메일 입력   │ │
│ └─────────────────┘ │
│ ┌─────────────────┐ │
│ │ 🔒 비밀번호      │ │
│ └─────────────────┘ │
│                     │
│ [    로그인     ]   │
│                     │
│ ─── 또는 ───        │
│                     │
│ [K] 카카오 로그인   │
│ [N] 네이버 로그인   │
│ [G] 구글 로그인     │
│                     │
│ 회원가입 | 비번찾기  │
└─────────────────────┘
```

## 🔄 사용자 플로우

### Flow 1: 신규 가입 → 첫 작품 열람
```
┌────────┐   ┌────────┐   ┌────────┐   ┌────────┐
│스플래시 │──►│ 로그인 │──►│취향설정│──►│  홈   │
└────────┘   └────────┘   └────────┘   └───┬────┘
                                           │
     ┌─────────────────────────────────────┘
     ▼
┌────────┐   ┌────────┐   ┌────────┐
│추천클릭│──►│작품상세│──►│첫화읽기│
└────────┘   └────────┘   └───┬────┘
                              │
                   ┌──────────┴──────────┐
                   ▼                     ▼
              ┌────────┐            ┌────────┐
              │다음화  │            │ 뷰어   │
              │(무료)  │            │ 종료   │
              └────────┘            └────────┘
```

### Flow 2: 유료 회차 구매

┌────────┐   ┌────────┐   ┌────────┐

└────────┘   └────────┘   └───┬────┘
     │                        │
     │ (토큰 있음)            ▼
     │                   ┌────────┐   ┌────────┐
     ▼                   │토큰충전│──►│결제완료│

└───┬────┘
    │
    ▼
┌────────┐
│뷰어열람│
└────────┘
```

## 📐 반응형 설계

### 모바일 (< 768px)
- 단일 컬럼 레이아웃
- 하단 네비게이션
- 터치 최적화

### 태블릿 (768px - 1024px)
- 2-3 컬럼 그리드
- 사이드 네비게이션 옵션

### 데스크톱 (> 1024px)
- 좌측 사이드바 네비게이션
- 3-4 컬럼 그리드
- 뷰어 양쪽 여백
