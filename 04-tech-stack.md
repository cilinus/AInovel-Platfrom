| 카테고리 | 기술 | 선정 이유 |
|----------|------|-----------|
| **런타임** | Node.js 20+ | JS 생태계, 비동기 처리 |
| **프레임워크** | NestJS | 구조화, DI, 모듈화 |
| **ORM** | Prisma | 타입 안전성, 마이그레이션 |
| **검증** | class-validator | 데코레이터 기반 검증 |

| **인증** | Passport + JWT | 소셜 로그인, 토큰 인증 |
| **캐시** | Redis | 세션, 캐싱, Rate Limiting |

### API 구조

```
NestJS 모듈 구조:
├── auth/           # 인증/인가
├── users/          # 사용자 관리
├── works/          # 작품 관리
├── episodes/       # 회차 관리
├── payments/       # 결제/토큰
├── ai/             # AI 연동
├── admin/          # 관리자
└── common/         # 공통 유틸
```

## 🗄️ 데이터베이스

### 주 데이터베이스

| 카테고리 | 기술 | 용도 |
|----------|------|------|
| **RDBMS** | PostgreSQL 15+ | 사용자, 작품, 결제 등 |
| **NoSQL** | MongoDB | AI 생성 로그, 비정형 데이터 |
| **캐시** | Redis | 세션, 인기 작품 캐시 |
| **검색** | Elasticsearch | 작품 검색, 태그 검색 |

### 데이터 모델 (핵심)

```sql
-- 사용자
users
├── id (UUID)
├── email
├── password_hash
├── nickname
├── role (USER/ADMIN)
├── token_balance
└── created_at

-- 작품
works
├── id (UUID)
├── title
├── synopsis
├── cover_image_url
├── genre
├── status (ONGOING/COMPLETED)
├── is_ai_generated
└── created_at

-- 회차
episodes
├── id (UUID)
├── work_id (FK)
├── episode_number
├── title
├── content (TEXT / S3 URL)
├── price (토큰)
├── is_free
└── published_at

-- 구매 내역
purchases
├── id (UUID)
├── user_id (FK)
├── episode_id (FK)
├── token_used
├── type (RENT/OWN)
└── purchased_at

-- 토큰 거래
token_transactions
├── id (UUID)
├── user_id (FK)
├── amount (+/-)
├── type (CHARGE/USE/BONUS)
├── description
└── created_at
```

## 🤖 AI 연동

### 소설 생성

| 용도 | 기술/서비스 | 비고 |
|------|-------------|------|
| **텍스트 생성** | OpenAI GPT-4 / Claude 3 | 메인 생성 엔진 |
| **로컬 모델** | Llama 3 / Mistral | 비용 절감용 |
| **파인튜닝** | OpenAI Fine-tuning | 장르별 특화 |
| **임베딩** | text-embedding-3 | 추천 시스템 |

### 만화 생성

| 용도 | 기술/서비스 | 비고 |
|------|-------------|------|
| **이미지 생성** | Midjourney / DALL-E 3 | 고품질 이미지 |
| **캐릭터 일관성** | ComfyUI + ControlNet | 스타일 유지 |
| **로컬 생성** | Stable Diffusion XL | 비용 절감 |

### AI 파이프라인

```
┌────────────────────────────────────────────────────┐
│                   AI 생성 파이프라인                 │
├────────────────────────────────────────────────────┤
│                                                    │
│  1. 설정 입력                                       │
│     └─► 장르, 캐릭터, 세계관 파라미터               │
│                                                    │
│  2. 프롬프트 구성                                   │
│     └─► 장르별 프롬프트 템플릿 + 컨텍스트           │
│                                                    │
│  3. AI 생성 (소설)                                 │
│     ├─► GPT-4: 고품질 생성 (유료 콘텐츠)           │
│     └─► Llama 3: 비용 효율 (무료 콘텐츠)           │
│                                                    │
│  4. AI 생성 (만화)                                 │
│     ├─► 스토리 → 씬 분할                          │
│     ├─► 각 씬 → 이미지 프롬프트                   │
│     └─► Midjourney/SD → 이미지 생성              │
│                                                    │
│  5. 후처리                                         │
│     ├─► 품질 검수 (금칙어, 정책)                   │
│     ├─► 포맷팅                                    │
│     └─► 저장 (DB + S3)                           │
│                                                    │
└────────────────────────────────────────────────────┘
```

### AI 비용 최적화

| 전략 | 설명 |
|------|------|
| 하이브리드 모델 | 무료 콘텐츠는 오픈소스, 유료는 상용 |
| 배치 생성 | 비인기 시간대 대량 생성 |
| 캐싱 | 인기 설정 조합 결과 캐싱 |
| 압축 프롬프트 | 토큰 사용량 최적화 |

| 서비스 | AWS | 용도 |
