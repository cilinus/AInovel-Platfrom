# 08. 모바일 웹앱 설계

## 📱 개요

> **"손 안에서 펼쳐지는 무한한 이야기"**

모바일 환경에 최적화된 PWA 기반 웹앱으로, 이북 리더 수준의 독서 경험을 제공합니다.

### 핵심 목표
- 📖 **이북 리더급 독서 UX** - 킨들, 리디북스 수준의 편안한 읽기 경험
- 📴 **오프라인 지원** - 지하철, 비행기에서도 끊김 없이
- 👆 **제스처 기반 조작** - 한 손으로 모든 조작 가능
- ⚡ **PWA** - 설치 없이 앱 같은 경험

## 🎯 모바일 타겟 사용자

### 주요 사용 시나리오

| 시나리오 | 상황 | 요구사항 |
|----------|------|----------|
| 출퇴근 독서 | 지하철/버스, 한 손 사용 | 한 손 조작, 흔들림 대응 |
| 취침 전 독서 | 어두운 방, 누워서 | 다크모드, 블루라이트 차단 |

| 오프라인 독서 | 지하철, 비행기 | 다운로드, 오프라인 모드 |

```

│    [1탭]         [1탭]         │
```

| 제스처 | 영역 | 동작 | 우선순위 |
|--------|------|------|----------|

| 우→좌 스와이프 | 전체 | 다음 페이지 | P0 |

| 더블 탭 | 전체 | 빠른 북마크 | P1 |

```
│  ├─ 추천 피드                                  │
```
