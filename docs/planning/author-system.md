# 09. 작가 시스템

## 📝 개요

본 플랫폼은 **AI 콘텐츠**와 **인간 작가 콘텐츠**가 공존하는 하이브리드 플랫폼입니다.
누구나 작가로 등록하여 자유롭게 연재할 수 있습니다.

```
┌─────────────────────────────────────────────────────────────┐
│                    콘텐츠 생태계                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   ┌─────────────────┐         ┌─────────────────┐          │
│   │   AI 콘텐츠     │         │  인간 작가 콘텐츠  │          │
│   │                 │         │                 │          │
│   │ • 자동 생성     │         │ • 자유 연재      │          │
│   │ • 무한 스케일   │         │ • 창작의 깊이    │          │
│   │ • 개인화       │         │ • 팬덤 형성      │          │
│   └────────┬────────┘         └────────┬────────┘          │
│            │                          │                    │
│            └──────────┬───────────────┘                    │
│                       ▼                                    │
│            ┌─────────────────┐                             │
│            │   독자 경험     │                             │
│            │  (원스톱 플랫폼) │                             │
│            └─────────────────┘                             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 📋 작가 등록 프로세스

### 등록 절차

```
[1단계: 신청]     [2단계: 인증]     [3단계: 정보입력]    [4단계: 완료]
    │                  │                  │                  │
    ▼                  ▼                  ▼                  ▼
┌─────────┐       ┌─────────┐       ┌─────────┐       ┌─────────┐
│작가 신청 │  →   │본인 인증 │  →   │계좌 등록 │  →   │작가 활동 │
│ 양식 작성│       │(휴대폰) │       │세금 정보 │       │  시작   │
└─────────┘       └─────────┘       └─────────┘       └─────────┘
                                                        (1-2일)
```

### 1단계: 신청 정보

| 항목 | 필수 | 설명 |
|------|------|------|
| 필명 | O | 활동할 작가명 |
| 이메일 | O | 연락용 이메일 |
| 장르 | O | 주력 장르 선택 (복수 가능) |
| 자기소개 | O | 100자 이상 소개글 |
| 포트폴리오 | - | 기존 작품 링크/파일 |
| SNS | - | 개인 채널 연동 |

### 2단계: 본인 인증

| 인증 항목 | 방법 | 목적 |
|-----------|------|------|
| 휴대폰 인증 | SMS 인증 | 본인 확인 |
| 이메일 인증 | 인증 링크 | 연락처 확인 |
| 성인 인증 | 본인인증 앱 | 콘텐츠 정책 (19금 제외지만 보호자 연령 확인) |

### 3단계: 정산 정보

| 항목 | 필수 | 설명 |
|------|------|------|
| 실명 | O | 세금 처리용 |
| 주민등록번호 | O | 원천징수 신고 |
| 정산 계좌 | O | 본인 명의 계좌 |
| 세금 유형 | O | 사업자/개인 선택 |

## 📖 작품 등록 시스템

### 작품 생성

```
┌─────────────────────────────────────────────────────────────┐
│                     작품 등록 화면                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  📚 기본 정보                                                │
│  ├─ 제목: [                    ]                            │
│  ├─ 장르: [로맨스 ▼] [판타지] [현대]                         │
│  ├─ 태그: #계약연애 #재벌 #달달                              │
│  └─ 연재 주기: [주 3회 ▼]                                   │
│                                                             │
│  📝 시놉시스                                                 │
│  ┌─────────────────────────────────────────┐                │
│  │ 작품 소개를 입력하세요...                 │                │
│  │                                         │                │
│  └─────────────────────────────────────────┘                │
│                                                             │
│  🖼️ 표지 이미지                                             │
│  [직접 업로드] [AI 생성 요청]                                 │
│                                                             │
│  💰 가격 정책                                                │
│  ├─ 무료 회차: [5] 화                                        │
│  └─ 유료 회차: [2] 토큰/화                                   │
│                                                             │
│                    [임시저장]  [등록하기]                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 작품 유형

| 유형 | 설명 | 가격 정책 |
|------|------|-----------|
| **무료 연재** | 전 회차 무료 | 광고 수익만 |
| **부분 유료** | 초반 무료 + 이후 유료 | 혼합 |
| **완전 유료** | 1화부터 유료 | 토큰 결제 |
| **선공개** | 구독자 선공개 | 일반 N일 후 공개 |

### 장르 분류

| 대분류 | 세부 장르 |
|--------|-----------|
| 로맨스 | 현대, 판타지, 역사, BL, GL |
| 판타지 | 정통, 현대, 퓨전, 게임판타지 |
| 무협 | 정통무협, 신무협, 현대무협 |
| 현대물 | 학원, 스포츠, 일상, 드라마 |
| 미스터리 | 추리, 스릴러, 호러 |
| SF | 사이버펑크, 우주, 포스트아포칼립스 |

> ⚠️ **성인물(19금) 제외**: 본 플랫폼은 전연령 콘텐츠만 취급합니다.

## 📜 작가 정책

### 금지 사항

- ❌ 타인 저작물 무단 게재
- ❌ 성인물/19금 콘텐츠
- ❌ 혐오/차별 콘텐츠
- ❌ 타 플랫폼 독점 작품 이중 게재
- ❌ AI 100% 생성물을 인간 창작으로 위장

### 작가 권리

- ✅ 작품 저작권 100% 작가 소유
- ✅ 언제든 연재 중단/삭제 가능
- ✅ 타 플랫폼 동시 연재 허용 (비독점)
- ✅ 출판/영상화 권리 협의 가능
- ✅ 수익 배분 투명 공개
